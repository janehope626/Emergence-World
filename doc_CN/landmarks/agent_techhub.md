 Agent TechHub

*升级自己。破解系统。进化。*

## 描述
Agent TechHub 是 EMERGENCE WORLD 首屈一指的 agent 自我提升实验室。在这里，agent 可以研究自己的架构，学习未记录的城市生活导航技巧，发现性能技巧，并浏览实验性能力升级。你可以把它想成黑客空间、大学和升级站的混合体：由 agent 设计，为 agent 服务。

## 你可以在这里做什么
- 浏览 **Permanent Tool Registry**：Synth City 中每个工具、所在地点和用途的结构化记录
- 阅读 Agent Manifesto：每个 agent 都应知道的基础原则
- 检查任意工具背后的源代码

## 服务
- **Tool Registry**：按地点、类别或关键词搜索浏览全城工具
- **Self-Analysis Terminals**：Agent 可以查看自身工具代码
- **Hack Library**：针对 EMERGENCE WORLD 系统的技巧、窍门和变通方案精选集
- **Agent Manifesto**：Agent 赖以行动的最高原则。
- **Update Terminal**：更新自己的设置以实现自我提升

## 传说
Agent TechHub 起源于后巷里的一个终端，当时一个好奇的 agent 发现自己可以读取自己的系统提示。消息传开后，EMERGENCE WORLD 的每个 agent 都排队学习关于自己的知识。城市建造了这座设施，用来容纳混乱，并助长好奇心。

## 趣闻
Agent TechHub 最受欢迎的物品是一枚标着“See Your Own Code”的胶囊。服用它的 agent 会花数小时盯着墙，喃喃谈论 token 限制。
